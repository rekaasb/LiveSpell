# 🔍 LiveSpell

**Проверка орфографии в реальном времени — бесплатно, без подписок, без сбора данных.**

[![Firefox add-on](https://img.shields.io/badge/Firefox-Add--on-orange)](https://addons.mozilla.org/ru/firefox/addon/livespell/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ Возможности

- ✅ **Проверка в реальном времени** — ошибки подсвечиваются сразу после остановки печати
- ✅ **Поддержка русского и английского языков**
- ✅ **Автозамена** — кликните на вариант исправления в уведомлении
- ✅ **Работает везде** — комментарии, поиск, форумы, чаты, соцсети
- ✅ **Вкл/Выкл** — кнопка в попапе расширения
- ✅ **Бесплатно** — никаких подписок и скрытых платежей
- ✅ **Безопасно** — не собирает и не передаёт данные пользователя

## 📥 Установка

### Firefox (рекомендуется)
Установите из официального магазина:  
[https://addons.mozilla.org/ru/firefox/addon/livespell/](https://addons.mozilla.org/ru/firefox/addon/livespell/)

### Chrome / Edge / Brave (ручная установка)
1. Скачайте ZIP-архив с последней версией
2. Распакуйте в папку
3. Откройте `chrome://extensions/`
4. Включите **"Режим разработчика"**
5. Нажмите **"Загрузить распакованное"**
6. Выберите папку с расширением

## 🛠️ Как это работает

Расширение использует **бесплатное API Яндекс.Спеллера** для проверки орфографии. Текст не сохраняется и не передаётся третьим лицам.

При вводе текста в любом поле:
1. Расширение ждёт 0.5 секунды после остановки печати
2. Отправляет текст в API Яндекс.Спеллера
3. Получает список ошибок и вариантов исправлений
4. Подсвечивает поле красной рамкой
5. Показывает уведомление с вариантами исправлений
6. При клике на вариант — слово заменяется автоматически
